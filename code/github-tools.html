                                  # 技術是第一生產力

                                  • 在當今快節奏的世界中，技術已經成為提高生產力的基石。它改變了我們工作、溝通和協作的方式。開源軟件以其透明和協作的特性成為這種轉變的推動力。像 GitHubopen in new window 這樣托管數百萬開源項目的平台，創造了一個充滿活力的生態系統，讓開發人員可以分享他們的想法並協作開發代碼。

                                  • 通過利用開源和 GitHubopen in new window 這樣的平台，開發人員能夠更有效地工作，減少開發時間，構建更好的軟件。

                                  • 大家可以盡情地在 issues 上給作者提出寶貴的建議和意見，或是提交 BUG 幫助作者完善項目，或是成為一些初學者（小白）們諮詢大佬的平台，極大地縮短了軟件開發的時間成本，也解決了人們面對互聯網工具資源匱乏的窘境。

                                  • 在 GitHub 上8年的學習、 git 中，大部分時間都在潛水，或是找一些必要資源。從最開始的無從下手，到現在基本上能熟練地在本地和伺服器搭建基本運行環境，熟練掌握 git 方法，將大佬們的項目 為我所用 。這期間我使用了無數的、各個平台上的項目、軟件，有時候甚至會為了一個簡單地環境搭建彻夜難眠、通宵達旦。也許這就是代碼的魅力、開源的魅力。

                                  • 今天就來簡單記錄一些我使用過的 神一般的生產力工具 ，方便大家更好地工作、學習。
